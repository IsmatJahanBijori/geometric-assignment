My Interview Blog

1. What is DOM and what is the purpose of DOM?

DOM refers Document Object Model which can interact with the connecting pages using programming languages. It defines logical structure of a document by manipulating. Purpose of DOM to use in events, methods, properties of HTML elements to change the interface of a web page.

2. How will you select HTML elements using DOM? Name the DOM methods.

There are five types of way to select HTML elements using DOM. Access those methods with the document object.

a) getElementsByTagName(): we use this method by tag name like h1, li, section etc.
b) getElementsByClassName(): using this method when we create one or many class name in html body to access the one or more elements.
c) getElementById(): we create an id in html body to access specific element for manipulating the logical structure.
d) querySelector(): it returns only the first element of selected element.
e) querySelectorAll(): it returns all the seleceted elements of the same className or id.

3. What is event bubble?

Event bubble defines the situation when an element gets an event and then it goes to its parent node. It is default behivour and event.stopPropagation() method is used to stop the event flow from the event target.

4. What is a callback function and why will you use it?

A callback function is a funtion that is passed into another function passing by as an argment.
